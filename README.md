Este é o repositório de código e ficheiros pertinentes ao trabalho do Grupo 1 de Laboratórios de Bioinformática (composto por **Ângela Sousa**, **António Araújo**, **Filipe Kuhlmann** e **Luís Pedrosa**).

O mesmo visa a análise de genes de *Klebsiella phage KP32* cujas proteínas estão involvidas na degradação da cápsula polissacarídea de *Klebsiella pneumoniae*, uma bactéria multirresistente.

A análise dos resultados aqui presentes está disponível em: https://padlet.com/asgdes/grupo-1-trabalho-de-laborat-rios-de-bioinform-tica-9p335t6w2qe4pp6j
